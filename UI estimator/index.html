Algorithm Details

Sequential Approach

Generate N random (x,y) points in range [0,1]. Count points where x²+y²≤1. Estimate π using 4*(points inside circle)/(total points).

Parallel Approach

Split work across multiple threads/workers. Each processes a batch of points independently. Aggregate results for final π estimation.
